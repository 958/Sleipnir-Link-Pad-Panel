Link Pad by958

■概要

　●ネスケ 9 の LinkPad もどき spx プラグイン

■動作確認環境

　●Windows XP SP2
　●Sleipnir 2.5.13
　●Internet Explorer 7

■インストール

　●丁寧にまとめて頂いてあるので、こちらを参考に
　　・Browser.js 第11回 Sleipnirのカスタム：パネルの追加 6（LinkPad）
　　　http://browserjs.blog102.fc2.com/blog-entry-290.html

　●plugins フォルダ、 resources フォルダを sleipnir のインストールフォルダに上書き
　●Sleipnir\plugins\panel\ の IEPanel.fx をコピペし、LinkPad.fx にリネーム
　●plugins_ini_editor（http://www.fenrir.co.jp/labs/plugins_ini_editor.html）などを使い、
　　plugins.ini の Panel セクションに LinkPad.fx を追加
　●Sleipnir2を再起動

　※古いバージョンがお好みの場合は、_oldフォルダ以下の spx ファイルを

　※UserAction スクリプトから追加できるようにする場合は↓

　●使用するスクリプトを Sleipnir\plugins\scripts\ にコピー

　　●AddLinkPadActiveElm.js
　　　・アクティブなエレメントがリンクあれば、そのリンクを LinkPad に追加
　　　　なければ、表示中のページをリンクパッドに追加
　　　・リンクをアクティブ（点線で囲まれるように）にする必要があるので、
　　　　ExtendedMenu をマウス割り当ての 左クリック→右クリック に割り当てている場合に ExtendedMenu に追加したり、
　　　　キーボードショートカットに追加したり、
　　　　マウス割り当ての 左クリック→右クリック に直接指定したりして使用する

　　●AddLinkPadMousePos.js
　　　・現在のマウスポジション下がリンクであれば、そのリンクを LinkPad に追加
　　　　なければ、表示中のページをリンクパッドに追加
　　　・マウス直下のリンクが対象となるため、ExtendedMenu では使用できない
　　　　マウスジェスチャやマウス割り当てで使用
　　　※一部のページで正常に動作しないっぽい

　　●AddLinkPadSelection.js
　　　・選択範囲内のリンクを全て LinkPad に追加
　　　　選択されていなければ、表示中のページをリンクパッドに追加
　　　・SmartSearch、ExtendedMenu など、いろいろなメニューに追加して使用可能だが、
　　　　選択する必要があるため、やや手間がかかる

　　●AddLinkPadAllTab.js
　　　・表示されている全てのタブを LinkPad に追加
　　　・メインメニュー、タブポップアップメニューなどに

　　●AddLinkPadNavigateTab.js
　　　・ナビゲーションロックされている全てのタブを LinkPad に追加
　　　・メインメニュー、タブポップアップメニューなどに

　　●AddLinkPadRightTab.js
　　　・現在のアクティブなタブより右側のタブを全て LinkPad に追加
　　　・メインメニュー、タブポップアップメニューなどに
　　　※タブの右クリメニューに追加した場合、右クリしたタブでは無く、
　　　　アクティブなタブより右側のタブが対象になるので、注意。

　　●AddLinkPadLeftTab.js
　　　・現在のアクティブなタブより左側のタブを全て LinkPad に追加
　　　・メインメニュー、タブポップアップメニューなどに
　　　※タブの右クリメニューに追加した場合、右クリしたタブでは無く、
　　　　アクティブなタブより左側のタブが対象になるので、注意。

■使用方法

　●リンクの追加
　　・リンクを LinkPad パネルに D&D すると一覧に追加される
　　・タブを D&D でもおｋ

　　・同伴の UserAction スクリプトでも追加可能

　●リンクを開く
　　・パネル上のリンクをクリックするとそのリンクを開く
　　　設定により、開いたリンクはパネル上から削除される
　　・Shift キーを押しながらクリックすると新規ウィンドウで開く設定とは反対の動作を行う
　　・Ctrl キーを押しながらクリックすると新規ウィンドウをアクティブにする
　　　（sleipnirの設定が新規ウィンドウはアクティブにする設定だった場合は、効かない）
　　・キーボードで Enter でも同様

　●リンクを消す
　　・パネル上のリンクを右クリックするとそのリンクをパネル上から削除
　　・キーボードで Delete でも同様

　●保存
　　・パネル上のアイテムが無い場所を右クリックするとメニューが表示される
　　　そのメニュー内の保存をクリック
　　・設定内容と、起動時に前回の状態を復元する設定の場合はリンク一覧を保存する
　　　後述の随時保存を使用したくない人用

　●設定変更
　　・パネル上のアイテムが無い場所を右クリックするとメニューが表示される
　　　設定内容は下記参照
　　・キーボードで アプリケーションキー でも同様

■設定

　●新規ウィンドウで開く
　　・チェックを入れた場合はリストをクリックするとリンクが新規ウィンドウで開く
　　　チェックを入れない場合はアクティブなウィンドウで開く

　●リンクを開いたらパネルから削除する
　　・チェックを入れた場合はリンクを開いたらパネルから削除する
　　・チェックを入れない場合はリンクを開いてもパネルから削除しない

　●起動時に前回状態を復元する
　　・チェックを入れた場合は終了時の状態を保存し、起動時に復元する

　●随時保存する
　　・プニルの強制終了対策用で、リンクパッドのデータに変更があった場合に随時保存する
　　　ただし、大量のリンクが収まっていたり、頻繁にリンクを出し入れする場合に、若干モッサリ感を感じるかも
　　　モッサリしてもいい人はチェックしとけばいい
